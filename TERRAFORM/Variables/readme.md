# Variables
In Terraform, a variable is a mechanism for accepting external values into a configuration, 
allowing you to parameterize your infrastructure code and make it more flexible and reusable

---

## The Terraform language uses the following types for its values:

**string:** a sequence of Unicode characters representing some text, like "hello".

**number:** a numeric value. The number type can represent both whole numbers like 15 and fractional values like 6.283185.

**bool:** a boolean value, either true or false. bool values can be used in conditional logic.

**list:** a sequence of values, like ["us-west-1a", "us-west-1c"].

**map:** a group of values identified by named labels, like {name = "Mabel", age = 52}.
